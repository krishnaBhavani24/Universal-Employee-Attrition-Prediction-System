# Universal-Employee-Attrition-Prediction-System
Employee Attrition Prediction System

A Machine Learning-based Employee Attrition Prediction System developed as a Capstone Project.
This project predicts whether an employee is likely to leave the organization based on various employee-related factors.

📌 Project Overview

Employee attrition is one of the major challenges faced by organizations.
This project uses Machine Learning models to analyze employee data and predict attrition probability.

The system is designed to be:

Generic
Scalable
Applicable to micro, medium, and large organizations
🎯 Objectives
Predict employee attrition using Machine Learning
Analyze factors affecting employee turnover
Compare multiple ML algorithms
Generate probability-based attrition insights
Help organizations improve employee retention

🧠 Key Insights
Employees with low salary and high overtime show higher attrition risk.
Job satisfaction and work-life balance strongly affect retention.
Random Forest performed better compared to other models.
